xCAD 框架提供了一种简单的方法来向各种 UI 元素添加自定义面板，例如：

* 任务窗格
* 特征管理器
* 模型视图
* 弹出窗口

[Windows Forms 控件](https://docs.microsoft.com/en-us/dotnet/api/system.windows.forms.usercontrol)和 [WPF 控件](https://docs.microsoft.com/en-us/dotnet/api/system.windows.controls.usercontrol)可以以相同的方式呈现。

参阅[表单和 WPF 控件示例](https://github.com/xarial/xcad-examples/tree/master/FormsAndWpfControls)，了解如何在同一控件托管在 SOLIDWORKS 不同面板中的演示。
