xCAD 框架提供了处理 SOLIDWORKS 第三方[存储（流）](stream)和[存储存储区](storage)中数据的功能（序列化和反序列化）。

关于第三方存储和存储存储区的详细概述，请参阅[使用 SOLIDWORKS API 在第三方存储中保存数据](https://www.codestack.net/solidworks-api/data-storage/third-party/)。

建议将此功能与[文档管理](/documents-management/)结合使用。

请参阅[问题管理器示例](https://github.com/xarial/xcad-examples/tree/master/IssuesManager)以了解如何使用自定义存储来存储和管理 SOLIDWORKS 中的文件特定问题列表。
